🗺️ ДЕТАЛЬНЫЙ ПРОМТ: ПЛАНИРОВЩИК МАРШРУТОВ НА ЯНДЕКС КАРТАХ
📋 ОБЩАЯ КОНЦЕПЦИЯ
Цель проекта
Создать интеллектуальный планировщик маршрутов, интегрированный с Яндекс.Картами, который позволяет пользователям создавать оптимальные маршруты с помощью ИИ-ассистента и переносить избранные места в планировщик.

Ключевые принципы
Интуитивность: Простой и понятный интерфейс
Персонализация: Адаптация под предпочтения пользователя
Интеллектуальность: ИИ-помощник для оптимизации маршрутов
Гибкость: Множество настроек и фильтров
🎨 ДИЗАЙН И UX/UI
Цветовая схема
Основной цвет: #2B5CE6 (синий Яндекса)

🔧 ТЕХНИЧЕСКАЯ РЕАЛИЗАЦИЯ
Архитектура системы
Frontend: React + TypeScript + Tailwind CSS
Карты: Яндекс.Карты API + Leaflet
ИИ: OpenAI GPT + собственная модель рекомендаций
Backend: Node.js + Express + MongoDB
Кэширование: Redis для быстрого доступа
Аналитика: Собственная система + Яндекс.Метрика
Производительность
Ленивая загрузка: Компоненты и данные по требованию
Виртуализация: Для больших списков мест
Кэширование: Маршруты и данные о местах
Оптимизация изображений: WebP + lazy loading
Service Workers: Офлайн функциональность

БАЗОВЫЙ УРОВЕНЬ (Бесплатный)
Цель: Показать ценность, создать привычку
Ограничения:
- 3 точки маршрута максимум
- 1 сохранённый маршрут
- 5 базовых рекомендаций в день
- Простые настройки (3 типа транспорта, 2 приоритета)

VIP УРОВЕНЬ (Платный)
Цель: Максимальная ценность и персонализация
Возможности:
- Неограниченное количество точек и маршрутов
- ИИ-оптимизация и персонализация
- Все настройки и тематики
- Экспорт, аналитика, офлайн-режим

ТОЧКИ КОНВЕРСИИ:
1. Попытка добавить 4-ю точку → модальное окно апгрейда
2. Попытка сохранить 2-й маршрут → предложение VIP
3. Запрос персонализированных рекомендаций → демо VIP-функций
4. Попытка экспорта → эксклюзивная функция VIP

UX СТРАТЕГИЯ:
- Базовый: "Попробуй и полюби" (простота, скорость)
- VIP: "Стань экспертом" (персонализация, возможности)

⭐ ИЗБРАННОЕ — УНИВЕРСАЛЬНОЕ ОКНО

- Вкладка "Метки": отдельные сохранённые точки (POI)
    - Можно добавить любую метку в текущий маршрут одним кликом
- Вкладка "Маршруты": сохранённые маршруты пользователя
    - Можно загрузить, переименовать, экспортировать, удалить маршрут

На странице планировщика всегда доступна кнопка "Избранное".
Пользователь может в любой момент добавить метку из "Избранного" в маршрут, не покидая страницу планировщика.

Такой подход:
- Не дублирует контент, а связывает пользовательские данные между разделами
- Делает работу с маршрутами и метками максимально гибкой и удобной
- Повышает ценность "Избранного" как личного инструмента пользователя
