НУ а если я изменю концепцию, то менять проект и саму суть я не собираюсь, но мне очень важно запустить его в таком виде: WayAtom — платформа для создания интерактивных путевых дневников с интеграцией карт и маршрутов по России.
 
 🚀 Концепция
 WayAtom — это платформа для создания интерактивных путевых дневников, в которых:
 
 📝 Текст объединяется с картами и маршрутами
 
 🗺️ Геолокация становится частью истории
 
 📊 Система реакций вместо комментариев
 
 🎯 Геймификация и достижения
 
 🎯 Основные функции
 📝 Путевые дневники
 Конструктор дневников с интуитивно понятным интерфейсом
 
 Геолокационная привязка контента к местам
 
 Интеграция с картами — автоматическое создание меток
 
 Интерактивные элементы — изображения, ссылки, маршруты
 
 Система черновиков и публикации
 
 Статистика просмотров и реакций
 
 🗺️ Навигация и карты
 Интерактивная карта с API Яндекс Карт
 
 Планировщик маршрутов с сохранением в базе данных
 
 Система меток с категориями и избранным
 
 Слои карты (стандартная, спутниковая, гибридная)
 
 Автоматическое масштабирование для маршрутов
 
 Геопривязка через Nominatim API
 
 📊 Система активности
 Типы активности пользователей
 
 Цветовая кодировка действий
 
 Статистика и аналитика
 
 Лента активности сообщества
 
 🛡️ Система модерации
 Автоматическая модерация контента
 
 Фильтры спама — проверка на запрещённые слова
 
 Геовалидация — проверка координат
 
 Панель администратора — ручная проверка контента
 
 🛠️ Технологии
 Фронтенд-стек

 Vite — быстрая сборка

 Карты и геолокация

 UI/UX
 Адаптивный дизайн
 
 Анимация и переходы

 🚀 Установка и запуск
 Клонируйте репозиторий:
 
 bash
 git clone https://github.com/simamakis/bestsite.git 
 cd best_site/frontend
 Установите зависимости:
 
 bash
 npm install
 Запустите проект:
 
 bash
 npm run dev
 🔧 Конфигурация
 Переменные окружения
 env
 VITE_API_BASE_URL=http://localhost:3002
 VITE_YANDEX_MAPS_API_KEY=ваш_ключ_yandex_maps
