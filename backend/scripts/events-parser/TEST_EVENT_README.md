# Тестовое событие для доработки UX

## Создание тестового события

Для создания тестового события выполните SQL скрипт:

```bash
# Вариант 1: Через psql
psql -U your_username -d your_database -f create-test-event.sql

# Вариант 2: Через pgAdmin или другой клиент
# Откройте файл create-test-event.sql и выполните его
```

## Что создается

Создается событие "Фестиваль 'День огурца' в Суздале" с:
- ✅ Полным описанием
- ✅ Координатами (Суздаль, Владимирская область)
- ✅ Датой и временем
- ✅ Категорией "Фестиваль"
- ✅ Хэштегами
- ✅ Организатором

## Использование для тестирования

После создания события:
1. Откройте календарь в приложении
2. Найдите событие в Суздале
3. Нажмите "Открыть детали"
4. Проверьте новое окно события в стиле glassmorphism

## Удаление тестового события

Если нужно удалить тестовое событие:

```sql
DELETE FROM events WHERE title = 'Фестиваль "День огурца" в Суздале';
```
