## 🎯 Ш Я  Ы:

### 1. Ы

### 2. Т

### 3. ТЫТ   VS CODE
- ctrl+p → ввести имя файла
- ctrl+g → ввести номер строки
- выделить проблемный код (обычно 5-10 строк)

### 4. СЬ COPILOT
- нажать ctrl+i (открыть copilot)
- ввести команду из todo_sonarqube.md
- пример: "исправь sql injection в этом коде, используя параметризованные запросы"
- просмотреть предложенные варианты (tab для переключения)
- выбрать лучший вариант (enter)

### 5.  СЯ
- прочитать изменённый код
- убедиться что логика не сломана
- проверить edge cases
- добавить комментарии если нужно

### 6. ТСТ
# если есть тесты:
npm test
# или
python -m pytest
# или запустить приложение и проверить вручную

### 7. СХ  ТТ
- ctrl+s сохранить файл
- обновить статус в todo_sonarqube.md
- в браузере отметить проблему как исправленную (если нужно)
- сделать коммит в git
